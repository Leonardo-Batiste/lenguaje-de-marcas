Ajustes del sistema

Aquí puedes cambiar cosas básicas del sistema. No afecta a los datos reales, solo es una simulación.

Alarmas

Puedes activar o desactivar las alarmas de glucosa alta o baja. Sirve para que el sistema avise si hay valores fuera del rango.

Rango objetivo

Rango recomendado: 80 a 140 mg/dL. Puedes modificarlo si quieres usar otros valores.

Modo de visualización

Modo claro o modo oscuro. Cambia los colores de la interfaz.

Idioma

Selecciona el idioma que prefieras. (Esta función no está implementada en la práctica, pero se pone aquí como ejemplo).

Sensor

Opción para conectar o cambiar el sensor. Si no funciona, revisa la conexión Bluetooth.

Información adicional

Recuerda que después de cambiar los valores tienes que comprobar que están guardados.

No pongas un rango de glucosa demasiado estrecho, porque el sistema puede avisar demasiado.

Si el sensor no manda datos, puede ser por interferencias o porque está lejos del dispositivo.

Estado de la batería

Aquí se muestra cuánta batería le queda al sensor y al dispositivo que recibe los datos.

Batería del sensor

Nivel aproximado: 65%. Si baja del 20% deberías cambiarlo pronto para que no deje de enviar datos.

Batería del dispositivo

Nivel del móvil o aparato donde se reciben los datos: 80%.

Consumo estimado

El sensor consume muy poca energía, pero si la señal es débil el consumo puede aumentar.

Consejos

Intenta mantener el sensor cerca del dispositivo para evitar pérdidas de conexión.

Información útil

El sensor suele durar entre 10 y 14 días, dependiendo del modelo.

Si notas que la batería baja muy rápido, podría ser un error del sistema.

Evita mojar el sensor si no está preparado para ello.
